# Browser SQLite 

An example setup for persisting SQLite in the browser with [sql.js](https://sql.js.org) and [origin private file system](https://developer.mozilla.org/en-US/docs/Web/API/File_System_API/Origin_private_file_system)

## Getting started

First, install the dependencies necessary.

```bash
npm install
```

Then, start the development server. This will auto-reload when files change

```bash
npm run dev
```
